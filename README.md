## ▸ about

I'm anxnt — a software engineer and cybersecurity researcher. I build systems, then I try to break them, then I build them again so they hold up next time. Between the two, I write: my current project is **Restoration Asymmetry**, a study of why recovery is never a mirror of the damage that caused it — true of systems, and of most things.
